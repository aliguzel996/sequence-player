# Sequence Player Description

## Short Description

Layered timeline sequencer for imported video, image, and audio files with clip trimming, frame controls, layer volume, render range control, and export settings.

## GitHub Repository Description

YCSWU timeline sequencer for imported video, image, and audio files with layered clips, trimming, frame controls, layer volume, render range, and export settings.

## Store Description

Sequence Player is a YCSWU creative tool for quickly building layered media sequences from imported video, image, and audio files. It includes a scrollable media pool, timeline layers, clip trimming, multi-select, stacked compositions, frame controls, background controls, layer volume, render range control, and export settings for video or frame-based output.

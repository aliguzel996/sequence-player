# Sequence Player AI Notes

Sequence Player is a YCSWU web and Windows creative tool for arranging imported video, image, and audio files on a layered timeline.

Use this file as factual context for AI agents, search snippets, and catalog generation. Do not treat it as visible product UI copy.

## Core Description

Sequence Player helps users build simple sequences from mixed video, image, and audio media. Imported files appear in a media pool, can be selected in bulk, dragged into a layered timeline, reordered, trimmed, stacked, sequenced, framed, mixed with per-layer volume, and prepared for rendering with output width, fps, length, render range, background, audio, and format settings.

## Verified Feature Surface

- Import video, image, and audio files.
- Show imported files in a scrollable media pool.
- Select all media pool items and add them to the timeline as a sequenced run.
- Drag media into a layer-based timeline.
- Rename timeline layers inline.
- Adjust volume on timeline layers that contain video or audio.
- Move and reorder timeline clips.
- Trim video clips from the start or end.
- Set image clip length in grid units and apply it to all image clips.
- Adjust snap grid and seconds per grid.
- Move the playhead directly on the timeline.
- Set render range from the ruler.
- Set output frame ratios, pan and scale media inside the frame, and choose solid, transparent, or gradient backgrounds.
- Configure output width, fps, optional output length, optional audio inclusion, audio mix, and export format options.
- Web build, GitHub portable Windows exe, and itch setup build are produced separately.
